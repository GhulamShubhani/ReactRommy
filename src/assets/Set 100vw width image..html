  return (

  1. Wrap your application with the Router component:
javascript
import React from "react";
import { BrowserRouter as Router, Switch, Route, Link } from "react-router-dom";
import AboutUs from "./AboutUs";
import ContactUs from "./ContactUs";

const App = () => {
  return (
    <Router>
      <div>
        <nav>
          <ul>
            <li>
              <Link to="/about">About Us</Link>
            </li>
            <li>
              <Link to="/contact">Contact Us</Link>
            </li>
          </ul>
        </nav>

        <Switch>
          <Route path="/about">
            <AboutUs />
          </Route>
          <Route path="/contact">
            <ContactUs />
          </Route>
        </Switch>
      </div>
    </Router>
  );
};

export default App;

In this example, we have created two components for the "About Us" and "Contact Us" pages. We then wrap our application in a Router component and create two Link components to navigate to these pages.

The Switch component is used to render the appropriate component based on the current URL path. We define two Route components with a path prop that corresponds to the URL path we want to match. When the URL matches one of these paths, the corresponding component is rendered.

With this setup, you should now be able to navigate to the "About Us" and "Contact Us" pages in your application.
